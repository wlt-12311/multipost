# 宇之然 AI 营销引擎 — 完整产品方案

> 给独立开发者和一人公司使用的 AI 营销工作台
> 输入产品/项目 → 自动生成策略+内容+页面 → 展示在营销广场获得流量

---

## 一、产品定位

### 一句话
**「帮你把产品说清楚、卖出去」的 AI 营销平台**

### 三句话
1. **工具**：AI 生成营销策略 + 内容 + 页面
2. **资产**：用户获得一个可直接发布的个人/公司/产品主页
3. **广场**：所有用户页面汇集在营销广场，互相引流、抱团推广

### 不是
- ❌ 通用文案生成器（Jasper/Copy.ai）
- ❌ 落地页建站工具（Framer/Stunning.so）
- ❌ 个人主页链接合集（Linktree）

## 四、三层模块详解

### 第一层：AI 引擎

#### 4.1 策略引擎

```
输入：产品名 + 一句话描述 + 行业 + 目标用户（可选）
  ↓
AI 分析输出：
  ├─ 定位建议（一句话定位、差异化点）
  ├─ 目标受众画像
  ├─ 竞品概览
  └─ 建议推广渠道（含理由）
```

**深度层级**：
| 级别 | 输出 | 适用场景 |
|:---:|:-----|:--------|
| ⚡ 速览 | 定位句 + 3个卖点 + 推荐渠道 | 快速测试想法 |
| 📋 标准 | 定位 + 受众 + 竞品 + 渠道 + 内容日历 + 增长假设 | 准备发布 |
| 🎯 深度 | 全部以上 + 定价建议 + 冷启动计划 + 关键词规划 | 认真创业 |

#### 4.2 内容工厂

基于策略一次性生成全部渠道内容：

| 渠道 | 输出 | 用途 |
|:-----|:----|:----|
| Twitter/X | 发布Thread（5-10条） | 预热、发布 |
| LinkedIn | 产品发布长文 | 专业社区 |
| 知乎 | 产品介绍帖 | 国内社区 |
| Email | 发布邮件（subject + body） | 订阅用户 |
| ProductHunt | 标题+描述+评论回复模板 | 产品发布 |
| GitHub | README 完整草稿 | 开源项目 |

**新增**：广场推广文案（自适应生成一张"广场卡片"）

### 第二层：页面工坊

#### 三种模板

| 模板 | 适用 | 页面内容 |
|:----|:-----|:--------|
| 🧑 **个人页** `/p/{username}` | 独立开发者、创作者 | Bio + 作品集 + 社交链接 + 推荐合作 |
| 🏢 **公司页** `/p/{company}` | 小团队、SaaS产品 | 产品介绍 + 定价指引 + 联系方式 |
| 📦 **产品页** `/p/{product}` | 新产品发布 | 卖点 + 截图 + CTA + 定价 + 用户评价 |

#### 技术规范

**一套 CSS 响应式框架**：
```css
/* 共用设计令牌 */
:root {
  --bg-primary: #0f0c29;
  --bg-secondary: #1a1a4e;
  --accent: #667eea;
  --accent-alt: #764ba2;
  --text: #e0e0e0;
  --radius: 12px;
  --font: 'Inter', system-ui, sans-serif;
}

/* 页面类型差异化通过 data-page-type 控制 */
[data-page-type="personal"] { /* 卡片式布局 */ }
[data-page-type="company"]  { /* 宽幅式布局 */ }
[data-page-type="product"]  { /* 展示式布局 */ }
```

**多语言支持**：
- i18n JSON 结构
- 初始支持：中文、英文
- AI 自动翻译所有内容字段
- 用户手动编辑后可锁定不覆盖

**发布方式**：
1. 托管模式：`{username}.yuzhiran.cn` 或 `yuzhiran.cn/p/{username}`
2. 导出模式：下载静态 HTML/CSS 自部署
3. 自定义域名：Pro 版支持


- 联盟效果 = 每人获 N 倍展示位

```

  ↓

  ↓

  ↓
点击互推 → 进入另一个用户的产品页
  ↓
"我也想要一个这样的页面？"
  ↓
注册 → 创建自己的页面 → 内容也展示在广场
```

## 五、AI 网关集成

各层调用模型的成本策略：

| 任务 | 推荐模型 | 原因 |
|:----|:--------|:----|
| 策略深析 | Claude/GPT-4 | 需要深度推理 |
| 内容生成 | DeepSeek/GPT-4o-mini | 性价比高 |
| 页面生成 | DeepSeek | 够用，省钱 |
| 多语言翻译 | DeepSeek | 量大需便宜 |
| 广场搜索 | - | 不需要AI |

网关自动路由，用户不看后台，只感知到"快又便宜"。

免费版用户 BYOK → 直接调他们的 API key
付费版用户 → 走网关预配额度 → 网关赚差价

## 十、关键风险和应对

| 风险 | 概率 | 应对 |
|:----|:---:|:----|
| AI内容太模板化 | 高 | 迭代行业差异化 prompts |
| 用户觉得不如直接问ChatGPT | 中 | 强调"结构化+多管道一次性输出+广场曝光" |
| 广场初期内容少 | 高 | 我们自己先填充5-10个精品页做标杆 |
| 付费转化率低 | 中 | 免费版足够有用，网关额度是硬价值 |
| 互推联盟没人用 | 中 | 初期人工撮合，邀请第一批用户互相挂 |
